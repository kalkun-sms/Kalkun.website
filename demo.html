Demo

To see how is Kalkun looks like in real system, you can try it on our demo server. There are 3 version of the demo:

  • Latest released version (0.5)
  • Latest development version (master branch)
  • Latest development version (devel branch)

Then login with username: 'kalkun', password 'kalkun' (both without quotes).

Please NOTE that you can do anything on the demo, but please DO NOT change the kalkun user password. All data will be cleanup every midday and midnight.

Dummy data on this demo is generated with generatedata.com.

About the demo

This is not free SMS service, this demo is intended to you to see how's Kalkun looks like in real live system. In this demo, you can preview Kalkun before you install on your system.
